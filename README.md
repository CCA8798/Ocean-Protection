# 🐋 海洋守护者

> **这是一份初二年级的比赛作业** —— 代码水平有限，请多包涵 🙏

一个关于海洋保护的科普展示网站，包含文章、数据图表和小游戏。

## 技术栈

- **Express 5** + **Multer 2**（CommonJS）
- 纯前端 HTML/CSS/JS（无框架）
- 文件型 JSON 存储（无数据库）

## 快速开始

```bash
pnpm install
node server.js
# 访问 http://localhost:3000
```

## 功能

- 📰 海洋保护科普文章（Markdown 渲染 + 数据图表）
- 🎮 6 款海洋主题小游戏
- 📊 内嵌图表（柱状图 / 饼图 / 世界地图）
- 📑 文章目录导航

## 项目结构

```
public/       → 前端静态文件
data/         → JSON 数据存储
server.js     → API 服务
```
